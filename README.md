# TO DO APP

## Installation

```bash

# install php packages
$ composer install


# install node packages
$ npm install


# migrate database using sqlite
$ php artisan migrate

# Optional: Seeding the database
$ php artisan db:seed


# install vite server for front end
$ npm run dev


# install laravel server
$ php artisan serve

```
